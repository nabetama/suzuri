# Suzuri

A lightweight Markdown editor with Live preview for macOS and Windows, built with Tauri.

<img width="3024" height="1964" alt="image" src="https://github.com/user-attachments/assets/fb4df3dd-b5fe-4712-8519-97367b279be8" />

## Install

Download the latest release from the [Releases](https://github.com/nabetama/suzuri/releases) page.

## Development

### Install

```sh
pnpm install
pnpm tauri dev
```

### Build

```sh
pnpm tauri build
```
